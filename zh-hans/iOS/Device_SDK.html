Device_SDK

Overview

The main Function in SDK includes device management, multi-room music(MRM), music control and device control and so on.

Device management includes device online/offline, get the device object which used to communicate with the device; MRM includes room setup, dismiss, change room name etc.; Music control includes playview related information management, playlist management etc.; Device control is the common control function, like preset, alarm setup, OTA etc.
